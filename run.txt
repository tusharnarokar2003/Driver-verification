# Step-by-Step Guide to Run the Face Verification Server

You encountered an error because your virtual environment (`venv`) was corrupted or moved from another folder. You also tried to use a Command Prompt (`cmd`) command in PowerShell to delete it, which caused a parameter error.

Follow these exact steps in your PowerShell terminal to fix everything and run the project:

### Step 1: Delete the Corrupted Environment
First, you need to cleanly remove the broken virtual environment. Run this exact command in your terminal:
```powershell
Remove-Item -Recurse -Force venv
```

### Step 2: Create a Fresh Virtual Environment
Now, create a brand-new virtual environment linked properly to your current folder:
```powershell
python -m venv venv
```

### Step 3: Activate the Environment
You must activate it before installing packages:
```powershell
.\venv\Scripts\activate
```
*(You should see `(venv)` appear at the start of your terminal prompt)*

### Step 4: Install the Required Packages
Since this is a fresh environment, reinstall the requirements:
```powershell
pip install -r requirements.txt
```

### Step 5: Start the API Server
Finally, start your FastAPI server:
```powershell
uvicorn main:app --host 0.0.0.0 --port 8000
```
If you get another prompt asking about the model downloading, you might need to run `python download_model.py` before starting the server.

---

### Step 6: Test the Project
1. Open the interactive dashboard in your browser by going to:
   http://localhost:8000/docs
2. Expand the green `POST /verify` accordion.
3. Click "Try it out".
4. Upload two images.
5. Click "Execute" and look at the JSON response (`verified: true`).
